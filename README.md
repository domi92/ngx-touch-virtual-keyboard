# NgxTouchVirtualKeyboard

## Debug

Run in 2 terminal

```
ng build ngx-touch-virtual-keyboard --watch
```
AND
```
ng serve ngx-touch-virtual-keyboard-demo
```

## Build and deploy

Run in sequence:

+++ CHANGE VERSION +++

```
ng build ngx-touch-virtual-keyboard

cd dist\ngx-touch-virtual-keyboard

npm publish --tag beta

```
